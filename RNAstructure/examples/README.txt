This directory contains some examples for input file formats.

.seq: Sequence file format:
bmorivector.seq
ca5s.seq
ec5s.seq
ec16s.seq
ICAM-1.seq
ivslsu.seq
RA7680.seq
RD0260.seq
RD0500.seq

.shape: SHAPE data input format:
16S.shape

.fasta: Alternate (FASTA) sequence format:
bmorivector.fasta

.lis: Oligonucleotide list file, for oligoscreen input:
oligoscreen_example.lis

.conf: Configuration file for Dynalign:
sample.conf
