ct2dot

  • Contents
  • Index

ct2dot is used to generate a dot bracket structure file from a single structure within a CT file. The CT structure can be regenerated by using dot2ct.

USAGE: ct2dot <ct file> <structure number> <bracket file>

Required parameters:

<ct file> The name of a file containing the CT structure to convert.
<structure number> The number, one-indexed, of the structure to convert in the CT file.
<bracket file> The name of a dot bracket file to which output will be written.

Options which don't require added values:

-h, -H, --help Display the usage details message.

Options which require added values:

NONE
